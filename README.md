# Inspeccion de Racks

Aplicacion React para configurar racks industriales antes de iniciar una inspeccion.

## Ejecutar en desarrollo

```bash
npm install
npm run dev
```

La primera version incluye:

- Pantalla inicial con accion de nueva inspeccion.
- Formulario de configuracion del rack.
- Generacion visual del rack por bahias y niveles.
- Seleccion local de posiciones del rack.
- Estado en memoria, sin base de datos.
